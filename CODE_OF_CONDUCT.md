# Code of Conduct

## Our Standards

This project is a professional open-source environment. All participants are expected to:

- Communicate respectfully and constructively
- Accept feedback without hostility
- Focus criticism on ideas and code, not individuals
- Be welcoming to contributors of all experience levels

The following is not acceptable:

- Harassment, insults, or personal attacks
- Dismissive or condescending language
- Sustained disruption of discussions

## Scope

This code of conduct applies to all project spaces — GitHub Issues, pull requests, discussions, and any other official communication channels.

## Enforcement

Violations may be reported to the project maintainers. All reports will be reviewed and handled with discretion. Maintainers reserve the right to remove comments, close threads, or ban participants who violate these standards.
